# Proyecto Módulo 2: Validación y Operaciones de Datos

Este proyecto consiste en la resolución de dos retos de programación utilizando Python, enfocados en estructuras de control (if, elif, else) y lógica de validación.

## Descripción de los programas

1. **Longitud de una frase**: El programa solicita una palabra y valida que su longitud esté entre 4 y 8 caracteres. Si no cumple, informa cuántas letras sobran o faltan.
2. **Encuentra el cuadrante**: A partir de coordenadas X y Y (distintas de cero), el programa identifica en cuál de los cuatro cuadrantes del plano cartesiano se ubica el punto.
